Munky Clothing Website
Welcome to the Munky Clothing repository! This project features a stylish and modern clothing website designed by TravelMunky Studio. The website is a showcase of a hypothetical clothing brand, Munky Clothing, and highlights a sleek and contemporary design with a focus on user experience.

Overview
Munky Clothing is a fashion-forward clothing website that blends trendy designs with high-quality materials. The site aims to provide a seamless and engaging shopping experience, featuring:

Homepage: A visually striking landing page with a cinematic introduction to the brand.
Product Pages: Detailed pages showcasing various clothing items, each with high-quality images and descriptions.
About Us: Information about the brand and its mission.
Contact Page: A form for inquiries and a way to get in touch with the team.
Features
Responsive Design: Optimized for all devices, from desktops to mobile phones.
Interactive Elements: Engaging animations and transitions for an immersive user experience.
Smooth Scrolling: Implemented with Locomotive Scroll for a modern browsing feel.
Cinematic Animations: Text and icons animate in a visually appealing manner, thanks to GSAP and ScrollTrigger.
Technologies Used
HTML/CSS: Structure and styling of the website.
JavaScript: For interactivity and animations.
Locomotive Scroll: Smooth scrolling effects.
GSAP & ScrollTrigger: Advanced animations and scroll-based triggers.
Getting Started
To run this project locally, follow these steps:

Clone the Repository:


git clone https://github.com/tasughalan/munky-clothing.git

Navigate to the Project Directory:
cd munky-clothing
Open index.html in Your Browser.
